High-fidelity mobile e-commerce prototype
	LUVE – Mobile E-commerce UI Prototype

Designed a high-fidelity mobile prototype for a fashion and activewear brand. The goal was to create a clean, modern shopping experience optimized for mobile users, with a strong focus on visual branding, product discovery, and usability.

	Designed a mobile-first layout to reflect how users primarily shop on social media
	Used a product grid to improve scan ability and reduce cognitive load
	Prioritized best sellers to guide purchasing decisions
	Rounded buttons and soft colours to match the brand’s feminine identity



 Consistent product card layout

Decision:
- All products use the same card structure.

Reason:
- Consistency reduces the learning curve and helps users compare items easily.

User benefit:
- Users can scan products faster and make decisions more confidently.

Skincare E-commerce Mobile UI Prototype

	High-fidelity mobile UI designed for a skincare e-commerce platform. The design focuses on mobile-first product discovery, clear pricing, and promotional visibility to support conversion-driven shopping experiences.
Key contributions:
•	Designed a mobile-first layout optimized for thumb navigation
•	Created a consistent product card system for scalability
•	Prioritized best-selling products to guide user decision-making
•	Applied brand-aligned colour palettes and spacing for visual clarity
Tools: Figma


Pill-style navigation

Decision:
I used pill-style navigation for product categories.

Reason:
This pattern is familiar in e-commerce and allows categories to be quickly scanned and switched.

User benefit:
Users can move between sections with minimal effort and without losing their place.

2. Consistent product card layout

Decision:
All products use the same card structure.

Reason:
Consistency reduces the learning curve and helps users compare items easily.

User benefit:
Users can scan products faster and make decisions more confidently.
